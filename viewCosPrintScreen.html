View CoS

The full details of the COS are displayed below. From the options below, choose:

  • Status history to view the history of any status changes of the COS;
  • Activity history to view any activity reported against the migrant:
  • Transfer to transfer the COS to another user (where permissions allow);
  • Print to save or print a PDF of the COS (you must have a PDF viewer installed to view and print the COS); or
  • Back to return to the previous screen.

Please note, if you want to amend then assign a COS, you can do so by selecting the Create and assign function, then Find an existing single certificate
